# 運算式和運算子
## 運算式
kotlin中的敘述分為很多種型式，運算式是其中一種敘述。運算式是由運算子(operator)和運算元(operand)所組成; 運算元可以是變數或是數值，而運算子就是數學上的運算符號,如+,-,*,%等等。

```
val intA = num + 2
+ 和 = 就是運算子
intA 和 num就是運算元
```

 名稱 | 說明
 --- | ---  
運算子 | +,-,*,/
運算元 | 1+2的1和2
單元運算子 | -24的-
二元運算子 | 1+2的+
沒有三元運算子 | if (判斷式) 程式碼1 else 程式碼2


## 指定運算子
最簡單也是使用最多的運算子就是指定運算子，符號「=」功能就是將等號右邊的值傳給左邊的變數
kotlin在宣告時，如果一開始沒有給值就必需要明確宣告，但沒有給值代表建立了變數，但沒有初始化，所以變數還不可以使用，要直到變數給第一次值後，才有初始化此變數。後續才可以操控這個變數。

```
//變數 = 值;
//intA = 10;

fun main(){
    val first = 20
    val second = 35
    var sum = 0
    println("計算前sum的值是$sum")
    sum = first + second
    println("計算後sum的值是$sum")
}
```

## 算數運算子
算數運算子(arithmetic operator)在數學上經常會使用到，下表列出它們的成員

算數運算子|意義|
-----|----|
+| 加 |
-| 減 |
*|乘|
/|除|
%|餘|
++|遞增|
--|遞減|



## 比較運算子 
if敘述常常會使用到比較運算子，所以先簡單了解一下if敘述的用法。比較運算子會比較左、右2個運算式，比較結果正確會傳出true,若比較結果錯誤，就傳回false
 

運算子 | 說明 |
| --- | --- | 
| == | 內容相等 |
| != | 內容不相等 |
| > | 大於 |
| < | 小於 |
| >= | 大於等於 |
| <= | 小於等於 |
```
if(條件判斷)
	敘述


fun main(){
    val x = 5
    if (x > 0)
        print("x的值大於零");
}
``` 





## 邏輯運算子 
邏輯運算子通常是結合多個比較運算式來綜合得到最終比較結果，用於較複雜的比較條件

運算子 | 說明 |
| --- | --- | 
| ! | not |
| &&  | and | 
| //(直的) | or |

```

fun main(){
    print("請輸入學生分數(0-100):");
    val num = readLine()?.toIntOrNull() ?: -1

    if(num < 0 || num > 100){
        println("學生分數輸入錯誤");
    }

    if(num < 60 && num >= 0){
        println("學生成績不及格");
    }

}
```

## 沒有三元運算式  

```
kotlin沒有三元運算式
語法:
if(條件判斷) 程式碼1 else 程式碼2 ;

```

```
fun main(){
    //kotlin沒有三元運算式，使用if運算式代替
    val in1:Double
    val in2:Double
    val in3:Double
    var max:Double

    print("請輸入第一個數:");
    in1 = readLine()?.toDoubleOrNull() ?: 0.0

    print("請輸入第二個數:");
    in2 = readLine()?.toDoubleOrNull() ?: 0.0

    max = if (in1 > in2) in1 else in2

    print("請輸入第二個數:");
    in3 = readLine()?.toDoubleOrNull() ?: 0.0

    max = if (max > in3) max else in3

    println("輸入三個數中最大的數為:$max")

}
```

## 數學算式轉換電腦運算
```
	// Name : ladder.kt
	//讓使用者輸入梯形的上底、下底及高，程式會計算梯形的面積(上底加下底乘以高除以2)
	
fun main(){
    val top:Int
    val bottom:Int
    val height:Int
    val area:Double

    println("請輸入梯形的上底(公分):")
    top = readLine()?.toIntOrNull() ?: 0

    println("請輸入梯形的下底(公分):")
    bottom = readLine()?.toIntOrNull() ?: 0

    println("請輸入梯形的高(公分):")
    height = readLine()?.toIntOrNull() ?: 0

    area = (top + bottom) * height / 2.0
    println("梯形的面積:$area 平方公分")
}

```
